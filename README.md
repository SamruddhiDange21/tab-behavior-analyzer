# Tab Behavior Analyzer

Analyze your browsing habits and understand how much time you spend on productive vs distraction-based websites.

---

##  Overview

This project is a simple data science dashboard built using **Python** and **Streamlit** that helps users visualize their tab usage behavior.

It categorizes websites into:

*  Productive (e.g., coding, learning)
*  Distraction (e.g., social media, streaming)

The goal is to provide insights into how users spend their time online and identify areas for improvement.

---

##  Features

*  Time spent per website (bar chart)
*  Productivity vs Distraction analysis (pie chart)
*  Total time and productive time metrics
*  Top distraction websites
*  Interactive dashboard using Streamlit

---

##  Tech Stack

##  How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/yourusername/tab-behavior-analyzer.git
cd tab-behavior-analyzer
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
python -m streamlit run app.py
```
